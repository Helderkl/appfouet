# Fouet Gestta — ERP para Confeitarias 🧁

Sistema de gestão inteligente para confeitarias brasileiras, com PDV de alta performance, controle de estoque, gestão financeira, catálogo com precificação automática, e integração com NFC-e via Focus NFe API.

## Stack Tecnológica

- **Frontend:** React.js + TypeScript + Vite
- **Estilização:** Tailwind CSS v4 (Glassmorphism + Dark accents)
- **Estado:** Zustand + TanStack Query
- **Gráficos:** Recharts
- **Ícones:** Lucide React
- **Host:** Netlify

## Módulos

| Módulo | Descrição |
|--------|-----------|
| Dashboard | KPIs do dia, gráfico semanal, alertas financeiros |
| Vendas / PDV | PDV de toque rápido, Kanban de pedidos, integração plataformas |
| Catálogo | Ficha técnica, precificação inteligente com margem de lucro |
| Estoque | Insumos, alertas de segurança, entrada de faturas |
| Financeiro | Fluxo de caixa, contas a pagar/receber, relatório de lucratividade |
| Funcionários | Colaboradores, permissões, integração folha de pagamento |
| Configurações | Dados da empresa, fiscal, certificado A1, NFC-e |
